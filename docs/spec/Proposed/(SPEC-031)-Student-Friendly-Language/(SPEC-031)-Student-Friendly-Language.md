# Student-Friendly Language

## Problem

The site's tone is adult-professional: "AI-assisted analysis with full source transparency" is not how a high schooler navigates. Student briefings exist but are buried deep in the card grid. The research banner feels like legal fine print to a teenager.

## Proposed Solution

Options (not mutually exclusive): (a) Promote student briefings higher in the card order or add a "Students" quick-link in the hero. (b) Add a simplified language mode or student-specific landing variant. (c) Adjust the student briefing generation prompt to use more accessible language.
