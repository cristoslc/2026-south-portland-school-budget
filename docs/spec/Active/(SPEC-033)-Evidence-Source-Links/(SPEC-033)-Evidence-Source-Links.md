# Evidence Source Links

## Problem

The Evidence page describes three evidence pools with source counts and summaries, but doesn't link to the actual primary sources. Tom (Tax-Conscious Resident) and Linda (School Board Insider) want to verify claims by checking the original meeting packets and videos. The page says "check the district's board portal" but doesn't provide direct links.

## Proposed Solution

Add direct links to spsdme.org budget page, Vimeo meeting recordings, and Diligent/BoardDocs agenda portal on the Evidence page. Populate from the evidence pool manifests which already contain source URLs.
