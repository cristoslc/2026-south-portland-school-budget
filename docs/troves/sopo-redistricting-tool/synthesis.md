# Synthesis: South Portland Redistricting Tool

## Key Findings

The Data4Good tool models four closure cases: Brown, Dyer, Kaler, and Small. Skillin stays open. The other four schools have 1,020 seats for 1,013 K-4 students.

### Algorithm design (from source repo)

The zone assignment uses a three-step process in `src/assignment.py`:

1. **Seed and grow** - Each school starts from a seed block and expands by drive distance.
2. **Fix capacity** - Schools that are too full give up blocks to schools that have space.
3. **Smooth routes** - Non-walkable neighbors stay together when possible.

The model uses OSMnx road data. It does not use straight-line distance. The walk limit is 0.75 miles. Roads and rails can block walks.

### Webapp features

The React/Vite webapp lets users click a census block and reassign it. Enrollment stats update live. Users can export zones as GeoJSON and upload other zone files.

### Block-level enrollment source

The trove includes the raw `data/student_blocks.json` file from the GitHub fork. It keeps the block-level enrollment schema and supports transport analysis.

## Points of Agreement (across sources)

- Skillin cannot close. The other four schools barely cover enrollment.
- The model uses Census 2020 blocks, not address-level rolls.
- The walk limit is 0.75 miles, based on road distance.
- The zones are generated by algorithm, not by hand.
- PreK at Dyer and Kaler changes the capacity math.
- All four scenarios fit capacity, but none are fully contiguous.

## Points of Disagreement (webapp vs raw model)

- The webapp shows "27% change schools" for Close Kaler and "34% change schools" for Close Dyer, but the raw model ties them.
- The webapp's per-school counts differ a bit from the model because of PreK overhead.

## Gaps

- **No contiguous zones** - all scenarios have island blocks.
- Full PreK expansion is not modeled.
- The raw block-level student export is ingested as `data/student_blocks.json`.
- No cost analysis is included.
- No demographic or equity analysis is included.
- No closure timeline or phase plan is included.
- The Census 2020 base is now 6 years old.
- There is no direct comparison with the district's nine workshop options.
